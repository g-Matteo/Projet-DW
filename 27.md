* Mise en place de l'index Bitmap Join
```sql
CREATE BITMAP INDEX BJI_Fact_Pays 
ON Fait_Evenement_Abonnement(G.Pays)
FROM Fait_Evenement_Abonnement F, Geographie G
WHERE F.Geo_Key = G.Geo_Key;
CREATE BITMAP INDEX BJI_Fact_CSP
ON Fait_Evenement_Abonnement(P.Categorie_socio_pro)
FROM Fait_Evenement_Abonnement F, Profil P
WHERE F.Profil_Key = P.Profil_Key;
```
