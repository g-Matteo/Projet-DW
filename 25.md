* Requête 1 : Chiffre d'affaires mensuel par pays
```sql
SELECT 
    D.Mois_Annee,
    G.Pays,
    SUM(F.Montant_Facture) AS Chiffre_Affaires_Total
FROM Fait_Evenement_Abonnement F
JOIN Date D ON F.Date_Key = D.Date_Key
JOIN Geographie G ON F.Geo_Key = G.Geo_Key
GROUP BY D.Mois_Annee, G.Pays;
```

* Requête 2 : Évolution du MRR par catégorie d’abonnement
```sql
SELECT 
    D.Mois_Annee,
    A.Nom_Plan,
    SUM(F.Delta_MRR) AS Net_New_MRR
FROM Fait_Evenement_Abonnement F
JOIN Date D ON F.Date_Key = D.Date_Key
JOIN Abonnement A ON F.Abonnement_Key = A.Abonnement_Key
GROUP BY D.Mois_Annee, A.Nom_Plan;
```

* Requête 3 : Analyse de l'attrition par profil démographique
```sql
SELECT 
    D.Annee,
    P.Tranche_Age,
    P.Categorie_socio_pro,
    COUNT(*) AS Nb_Resiliations
FROM Fait_Evenement_Abonnement F
JOIN Date D ON F.Date_Key = D.Date_Key
JOIN Profil P ON F.Profil_Key = P.Profil_Key
JOIN Type_Evenement E ON F.Evenement_Key = E.Evenement_Key
WHERE E.Categorie_Evenement = 'Attrition' -- Filtre sur les résiliations
GROUP BY D.Annee, P.Tranche_Age, P.Categorie_socio_pro;
```

* Requête 4 : Volume de transactions par pays et par catégorie d’abonnement
```sql
SELECT 
    G.Pays,
    A.Nom_Plan,
    COUNT(*) AS Volume_Transactions
FROM Fait_Evenement_Abonnement F
JOIN Geographie G ON F.Geo_Key = G.Geo_Key
JOIN Abonnement A ON F.Abonnement_Key = A.Abonnement_Key
GROUP BY G.Pays, A.Nom_Plan;
```

* Requête 5 : Chiffre d'affaires par trimestre et profil
```sql
SELECT 
    D.Annee,
    D.Trimestre,
    P.Categorie_socio_pro,
    SUM(F.Montant_Facture) AS CA_Trimestriel
FROM Fait_Evenement_Abonnement F
JOIN Date D ON F.Date_Key = D.Date_Key
JOIN Profil P ON F.Profil_Key = P.Profil_Key
GROUP BY D.Annee, D.Trimestre, P.Categorie_socio_pro;
```

* Requête 6 : Taux d'échec de paiement par pays
```sql
SELECT 
    G.Pays,
    COUNT(*) AS Nb_Echecs_Paiement
FROM Fait_Evenement_Abonnement F
JOIN Geographie G ON F.Geo_Key = G.Geo_Key
JOIN Type_Evenement E ON F.Evenement_Key = E.Evenement_Key
WHERE E.Nom_Evenement = 'Echec_Paiement'
GROUP BY G.Pays
ORDER BY Nb_Echecs_Paiement DESC;
```
